# 🐄 Agro Forte & Futuro Sustentável: Simulador de Pecuária Rotacionada

## 🌾 Sobre o Projeto
Este projeto é uma aplicação web interativa desenvolvida exclusivamente para o **Concurso Agrinho 2026**. O objetivo do software é simular e demonstrar de forma matemática, prática e visual o impacto do **Manejo de Pastagem Rotacionada (Sistema Voisin)** na criação de gado de corte e leite.

A ferramenta resolve o dilema central do tema deste ano: **o equilíbrio entre produção e meio ambiente**. Através de dados configurados pelo usuário, o simulador prova que técnicas modernas de gestão de solo permitem aumentar a lucratividade do produtor (**Agro Forte**) enquanto regeneram a terra e sequestram carbono da atmosfera (**Futuro Sustentável**).

---

## 🎯 Alinhamento com o Tema do Agrinho 2026
* **Agro Forte (Produção):** O simulador demonstra que a divisão correta do pasto em piquetes aumenta a oferta de capim de alta qualidade nutricional. Isso eleva a taxa de lotação e faz o boi engordar mais rápido, gerando mais arrobas ($\text{@}$) por hectare/ano sem expandir a área de terra.
* **Futuro Sustentável (Meio Ambiente):** Ao respeitar o tempo de descanso do capim, as raízes crescem mais profundas, melhorando a biologia do solo, evitando a erosão e realizando o **sequestro de carbono**. Isso transforma a pecuária em uma aliada contra as mudanças climáticas (Pecuária de Baixo Carbono).

## 🛠️ Tecnologias Utilizadas
O projeto foi desenvolvido do zero, utilizando as tecnologias fundamentais da Web (sem a necessidade de bibliotecas externas), garantindo um código leve, rápido e semântico:

* **HTML5:** Para a estruturação semântica da página e acessibilidade dos dados.
* **CSS3:** Para a criação de um design moderno, responsivo (adaptável a celulares e computadores) e com identidade visual focada no campo (tons terrosos e verdes).
* **JavaScript (ES6):** Para a lógica de programação do simulador, cálculos matemáticos dinâmicos e animações das barras de progresso.

## 🖥️ Como Executar o Projeto Localmente

Para rodar este simulador no seu computador, você não precisa instalar nenhum programa ou servidor. Basta seguir os passos:

1. Baixe os arquivos do projeto para uma pasta no seu computador:
   * `index.html`
   * `style.css`
   * `script.js`
2. Certifique-se de que os três arquivos estão salvos **na mesma pasta**.
3. Dê dois cliques no arquivo `index.html`.
4. O simulador abrirá automaticamente no seu navegador de internet padrão.

## 📊 Regras de Negócio do Simulador

O algoritmo do sistema foi programado com base em lógicas reais de manejo de pastagens:
* **1 Piquete (Pecuária Extensiva Tradicional):** O gado pratica o superpastoreio, compacta o solo e consome o capim antes do tempo de recuperação. O resultado é baixa produtividade ($1\text{ @/ha/ano}$) e degradação ambiental crítica.
* **2 a 7 Piquetes (Transição):** Melhora o aproveitamento, mas o tempo de descanso da pastagem ainda é insuficiente para o potencial máximo da planta e do solo.
* **8 ou mais Piquetes (Manejo Rotacionado Intensivo):** O capim atinge o ápice nutricional, o solo acumula matéria orgânica, retém água e atua como um dreno de carbono. A produtividade atinge a eficiência máxima de até $15\text{ @/ha/ano}$.
